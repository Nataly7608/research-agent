# Research Agent — Инструкции

## Цель проекта
Research Agent — это CLI-агент для проведения исследований, сбора и анализа информации.

## Конфигурация
- Основной агент: `researcher` — исследователь (primary)
- MCP: Playwright — для веб-скрапинга
- Настройки в `opencode.json`

## Команды
- `npm run dev` — запуск проекта (когда будет реализован)
- `npm test` — запуск тестов
